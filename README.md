# Game Diary

GitHub Pages용 개인 게임 기록 웹페이지입니다.

## 수정 방법
대부분의 내용은 `data.js`만 수정하면 됩니다.

- 게임 제목과 소개: `profile`
- 등장인물: `characters`
- 챕터: `chapters`
- 사건 순서: `timeline`

## GitHub Pages 배포
1. 이 폴더의 파일을 GitHub 저장소에 업로드합니다.
2. 저장소의 `Settings` → `Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. Branch를 `main`, 폴더를 `/root`로 설정합니다.
5. 저장 후 생성된 주소로 접속합니다.
